Learning the process of science cannot be done overnight. Rather, just like a plumber or a carpenter has a toolbox full of the tools needed for their trade to get the job done, we will methodically gather the scientific tools needed to place in our scientific toolbox so that we can better understand the process of science, assess science in the media, and make informed conclusions and decisions based on the information presented.

The course runs over the session and has the following structure--all of which is accessible through ANGEL:
 
  • Module 01: Introduction
  • Module 02: The Scientific Method
  • Module 03: Descriptive Statistics
  • Module 04: Graphs
  • Module 05: Inferential Statistics
  • Module 06: Integration
  • Module 07: Science and Media
  • Module 08: Climate Change
  • Module 09: Pandemics
  • Module 10: Final Assessment- Group Project
