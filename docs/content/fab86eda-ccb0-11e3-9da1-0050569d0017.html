We will be using Google Docs and Spreadsheets throughout this course. These assignments begin in Module 4, however we have a required assessment in this module where you will practice using Google Docs.

Steps to create a Google account:

Step 1: Create Your Google Account Now

You will need to make sure you have a Google account set up so that you can access course documents. If you do not already have a Google account, go to Google's signup webpage to open one. Note: you do not need a "Gmail" account, just a "Google" account with your regular email address.
