Just like on-campus students, as a Penn State student you have a wealth of library resources available to you!

As a registered user of Penn State Libraries, you can...

  • search for journal articles (many are even immediately available in full-text)
  • request articles that aren't available in full-text and have them delivered electronically
  • borrow books and other materials and have them delivered to your doorstep
  • access materials that your instructor has put on electronic Library Reserves
  • talk to reference librarians in real time using chat, phone, and e-mail
  • ...and much more!

Activity

So that we can be sure you are able to take advantage of Library resources in this course, follow these steps NOW:

  1. Go to the Library Information for Off-site Users Web site
  2. Follow the link on that page to "Register with the Libraries" and complete the registration form.
  3. Learn more about obtaining library materials at a distance by checking out the information on the "Library Distance Learning Delivery" page.
